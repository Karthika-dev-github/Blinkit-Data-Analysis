# Blinkit Data Analysis Project

## Overview
This project analyzes Blinkit sales data using Python to identify trends, patterns, and business insights.

## Tools & Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Tasks
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Insight Generation

## Key Insights
- Identified sales trends across categories
- Analyzed customer purchasing patterns
- Visualized performance metrics
